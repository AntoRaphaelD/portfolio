Quick setup:
1. npm install
2. npm run dev
3. Build: npm run build

To deploy to Firebase Hosting:
- Install firebase-tools: `npm i -g firebase-tools`
- `firebase init hosting` and set `dist` (or `.`) accordingly after building.
- `firebase deploy`
